Чтобы запустить программу на телефоне достаточно скачать и установить apk файл на мобильном устройстве, 
он находится в директории mobile/apk и называется app-release.apk. Он доступен только для телефонов на базе 
андроида 11 и выше.

Если не устанавливается файл по различным причинам, можно через запустить android studio, собрать проект и запустить
по режиме отладки



Если вы хотите запустить сервер и базу данных на своём сервере(компьютере) достаточно иметь python 3.14(рекомендованаая)
и находясь в директории запустить виртуальную среду venv командой venv/Scripts/activate или linun source venv/Scripts/activate
после активации виртаульной среды можно запускать сервер командой python app.py или для линукса python3 app.py
при первом запуске сервера создаётся база данных и два пользователя в формате login:password 
admin1:1234 с ролью админа
owner1:1234 с ролью бизнесмена
После запуска сервера нужно собрать андроид приложение через android studio, заменив при этом в MainActivity 
адресс на ваш сервер. После собрать, установить и можно запускать и пользоваться приложением


Описание:
Это приложение является интерактивной картой локаций по днр, где гость может просматривать локации, описание и т.д.
так-же смотреть отзывы локаций
Авторизированный пользователь может добавлять в избранное локации, оставлять отзыв.
Админ/модератор может упрвалять локациями и пользователями, смотреть статистику, редактировать карточки локаций
Бизнесмен может управлять своей карточкой

Это не все функции которые доступны, ознакомтесь с ними в приложении
